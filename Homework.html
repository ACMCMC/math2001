  sorry

Homework 2

  1. Let \(x\) be a rational number whose square is 4, and which is greater than 1. Show that \(x=2\).

    Suggested steps: Prove that \(x(x+2)=2(x+2)\), then cancel to deduce that \(x=2\).

    /- 4 points -/
theorem problem1 {x : ℚ} (h1 : x ^ 2 = 4) (h2 : 1 < x) : x = 2 := by
  sorry

  2. Let \(s\) be a rational number for which \(3s ≤ -6\) and \(2s \ge -4\). Show that \(s=-2\).

    You will probably use the lemma le_antisymm, stating if \(x\le y\) and \(x\ge y\) then \(x = y\).

    /- 5 points -/
theorem problem2 {s : ℚ} (h1 : 3 * s ≤ -6) (h2 : 2 * s ≥ -4) : s = -2 := by
  sorry

  3. Let \(a\) and \(b\) be real numbers for which \(a = 2 - b\). Show that either \(a + b = 2\) or \(a + b = 8\).

    /- 2 points -/
theorem problem3 {a b : ℝ} (h : a = 2 - b) : a + b = 2 ∨ a + b = 8 := by
  sorry

  4. Let \(t\) be a rational number and suppose that \(t=-2\) or \(t=3\). Show that \(t^2-t-6=0\).

    /- 4 points -/
theorem problem4 {t : ℚ} (h : t = -2 ∨ t = 3) : t ^ 2 - t - 6 = 0 := by
  sorry

  5. Let \(x\) be any integer. Show that \(2x \ne 7\).

    You will probably use the lemma le_or_succ_le, stating that for integers \(x\), \(y\) either \(x\le y\) or \(x \ge y+1\).

    /- 5 points -/
theorem problem5 {x : ℤ} : 2 * x ≠ 7 := by
  sorry

  6. Let \(t\) be a real number for which \(t^3=t^2\). Show that either \(t=1\) or \(t=0\).

    You will probably use the lemma eq_zero_or_eq_zero_of_mul_eq_zero (stating that if \(ab=0\) then either \(a=0\) or \(b=0\)), as well as the cancel tactic.

    /- 5 points -/
theorem problem6 {t : ℝ} (ht : t ^ 3 = t ^ 2) : t = 1 ∨ t = 0 := by
  sorry

  1. Let \(a\), \(b\) and \(c\) be integers and suppose that \(a^2\) divides \(b\) and \(b^3\) divides \(c\). Show that \(a^6\) divides \(c\).

    /- 4 points -/
theorem problem1 {a b c : ℤ} (hab : a ^ 2 ∣ b) (hbc : b ^ 3 ∣ c) : a ^ 6 ∣ c := by
  sorry

  2. (transitivity rule for modular arithmetic) Let \(a\), \(b\), \(c\) and \(n\) be integers, and suppose that \(a\equiv b \mod n\) and \(b\equiv c \mod n\). Show that that \(a\equiv c \mod n\).

    Do this in the style of Example 3.3.3, Example 3.3.6, etc., that is, directly from the definitions.

    Note: the problem can also be solved in two lines as

    calc a ≡ b [ZMOD n] := h1
  _ ≡ c [ZMOD n] := h2

    The idea is not to do this.

    /- 4 points -/
theorem problem2 (h1 : a ≡ b [ZMOD n]) (h2 : b ≡ c [ZMOD n]) : a ≡ c [ZMOD n] := by
  sorry

  3. Let \(x\) be an integer for which \(x\equiv 2\mod 5\). Show that \((2 x + 3) ^ 2 \equiv (2 \cdot 2 + 3) ^ 2\mod 5\).

    Do this in the style of Example 3.3.11, that is by applying the relevant lemmas from Section 3.3 one by one.

    Note: the problem can also be solved in one line as rel [h]. The idea is not to do this.

    /- 4 points -/
theorem problem3 {x : ℤ} (h : x ≡ 2 [ZMOD 5]) :
    (2 * x + 3) ^ 2 ≡ (2 * 2 + 3) ^ 2 [ZMOD 5] := by
  sorry

Homework 6

  1. Let \(x\) be a real number. Show that \(x^2-3x+2=0\) if and only if \(x=1\) or \(x=2\).

  2. Let \(x\) be an integer. Show that if \(x\) is odd then so is \(x^3\).

    This problem appeared on Homework 3, but this time, re-do it by converting it to a modular arithmetic problem and then using the techniques of Section 3.4, as indicated in the first line.

theorem problem2 {x : ℤ} (hx : Int.Odd x) : Int.Odd (x ^ 3) := by

  sorry

  5. (hard) Show that \(\lnot\forall x, P(x)\) is logically equivalent to \(\exists x, \lnot P(x)\).

    You will need to use the law of the excluded middle (the tactic by_cases).

    The point of this exercise is that you are proving the lemma not_forall, from Table 5.1. So don’t use that lemma or the tactic push_neg which depends on it; instead prove this from scratch. You will need to use the law of the excluded middle.

    Write this problem only in Lean, not on paper.

    /- 5 points -/
theorem problem5 (P : α → Prop) : ¬ (∀ x, P x) ↔ ∃ x, ¬ P x := by
  sorry

  6. Show step by step using the rules in Table 5.1 that \(\lnot(\forall a b :\mathbb{Z}, ab=1 \to a = 1 \lor b = 1)\) is logically equivalent to \(\exists a b :\mathbb{Z}, ab = 1\land a \ne 1 \land b \ne 1\).

    /- 4 points -/
